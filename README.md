# T02_03_Grupo8_Backend_Hoteles

Proyecto backend para la gestión de hoteles desarrollado para la materia **Ingeniería de Software**.

El objetivo del proyecto es implementar una aplicación backend utilizando un framework de backend y siguiendo una estructura básica de modelo, repositorio, servicio y controlador.

---

## Descripción

La aplicación permite la gestión de información relacionada con hoteles, como el registro, consulta, actualización y eliminación de datos.  
El sistema expone servicios REST y cuenta con documentación mediante Swagger para facilitar las pruebas y el uso de la API.

---

## Tecnologías utilizadas

- Lenguaje: Java / Python
- Framework Backend: Spring Boot / FastAPI / Flask / Django DRF
- Base de datos: MySQL / SQLite
- Control de versiones: Git y GitHub
- Documentación de servicios: Swagger

---

## Estructura del proyecto

El proyecto está organizado de la siguiente manera:

src/
├── controller
├── service
├── repository
├── model


Esta estructura permite separar la lógica del negocio, el acceso a datos y la exposición de los servicios.

---

## Funcionalidades

- Gestión de hoteles (crear, listar, actualizar y eliminar)
- Servicios REST
- Documentación de endpoints con Swagger
